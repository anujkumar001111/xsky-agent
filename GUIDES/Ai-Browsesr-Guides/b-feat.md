## Browser Functionality

- Network
  - Browsers download various resources through the network module, such as HTML text, JavaScript code, stylesheets, images, audio and video files, etc.
  - The network layer is essentially very important because it is time-consuming and requires secure access to resources on the internet.
- Resource Management

  - Resources downloaded from the network or obtained locally need efficient mechanisms to manage them.
  - For example, how to avoid duplicate downloads, how to cache resources, etc.
