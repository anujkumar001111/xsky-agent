take control of my browser

Assume you are a data-science mentor.

1. On the Trending → New datasets page, scroll until five datasets < 24 h old appear.  
2. For each one: thumbs-up, leave a 15-word constructive comment, wait 15 s.  
3. If any Kaggle security warning appears, stop immediately.  
4. Report each up-voted slug.
